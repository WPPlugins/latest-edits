== Description ==

Adds a widget in your Dashboard which shows a list of the latest pages/posts edits, so you can easily find where you were working last. 

== Installation ==

1. Upload the plugin folder to the '/wp-content/plugins/' directory
2. Find the 'Plugins' menu in WordPress Admin and activate the plugin there.
3. Go to the Dashboard and find the widget with the latest edits.
